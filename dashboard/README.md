# Dashboard

Coloque aqui o arquivo `.pbix` e, se quiser, capturas de tela de cada página do relatório (formato `.png`), para quem visitar o repositório conseguir ver o resultado sem precisar abrir o Power BI Desktop.

Sugestão de nomes para os prints, seguindo as páginas do relatório:

- `01_carteira_de_credito.png`
- `02_concessao.png`
- `03_safra.png`
- `04_politica_de_credito.png`

Adicionar os prints diretamente no README principal (`![nome](./dashboard/01_carteira_de_credito.png)`) ajuda bastante quem só quer navegar pelo GitHub sem baixar nada.
